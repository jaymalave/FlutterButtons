# Glassmorphism Button
A custom glassmorphism button as a flutter package.




## Installation🔧

[Official Installation Guide](https://pub.dev/packages/glassmorphism_buttons/install)
## Documentation📚

**Class**   
GlassyButton

**Properties**
 - height?
 - width?
 - title
 - onTap
 
 **Example/Usage**
 ```dart
GlassyButton{
    height: 50,
    width: 200,
    title: 'Frosted Glass',
    onTap: (){
       print("I'm loving this button!😃")
    }
}
```
